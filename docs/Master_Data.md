# ข้อมูลหลัก (Master Data)

เอกสารนี้รวบรวมข้อมูลอ้างอิงพื้นฐาน (Master Data / Lookup Data) ที่ใช้งานในระบบ WongHiran Shop ทั้งฝั่ง Frontend และ Backend  
ข้อมูลเหล่านี้เป็น **ค่าคงที่ระดับระบบ** ที่ถูกนำใช้เป็น Dropdown, Filter หรือ Validation ในหลายส่วนของแอปพลิเคชัน

---

## 1. หมวดหมู่สินค้า (`Product Categories`)

**ใช้ในส่วน:** ฟอร์มเพิ่ม/แก้ไขสินค้า, Filter หน้าจัดการสินค้า, หน้าร้านลูกค้า  
**Field ในฐานข้อมูล:** `products.category` (varchar)

| ลำดับ | ชื่อหมวดหมู่ | คำอธิบาย |
| :---: | :--- | :--- |
| 1 | ของเล่นเด็ก | ของเล่นสำหรับเด็ก ของเล่นเสริมพัฒนาการ |
| 2 | อุปกรณ์กีฬา | สินค้าและอุปกรณ์เพื่อการออกกำลังกายและกีฬา |
| 3 | อุปกรณ์ทำความสะอาด | ผลิตภัณฑ์และเครื่องมือทำความสะอาดบ้านและสำนักงาน |
| 4 | เครื่องครัว | ภาชนะ อุปกรณ์ประกอบอาหาร และของใช้ในครัว |
| 5 | อุปกรณ์แคมปิ้ง | สินค้าสำหรับกิจกรรมกลางแจ้งและการตั้งแคมป์ |
| 6 | พลาสติก | สินค้าและบรรจุภัณฑ์ที่ทำจากพลาสติก |
| 7 | อุปกรณ์ไฟฟ้า | สายไฟ ปลั๊ก สวิตช์ และอุปกรณ์ระบบไฟฟ้า |
| 8 | เครื่องใช้ไฟฟ้า | เครื่องใช้ไฟฟ้าในครัวเรือนขนาดเล็ก |
| 9 | อุปกรณ์สัตว์เลี้ยง | ของใช้และอุปกรณ์สำหรับสัตว์เลี้ยง |
| 10 | เครื่องมือช่าง | เครื่องมือช่างและอุปกรณ์ซ่อมบำรุง |
| 11 | สินค้าเทศกาล | สินค้าตามฤดูกาลและเทศกาลต่างๆ |
| 12 | เซรามิค | ภาชนะ ของตกแต่ง และสินค้าเซรามิค |
| 13 | อุปกรณ์ขายสินค้า | อุปกรณ์สำหรับผู้ค้าส่ง เช่น ถุง กล่อง ป้ายราคา |
| 14 | ของใช้ในบ้านและเฟอร์นิเจอร์ | เฟอร์นิเจอร์ขนาดเล็ก และของตกแต่งบ้าน |
| 15 | เบ็ดเตล็ด | สินค้าอื่นๆ ที่ไม่อยู่ในหมวดหมู่ข้างต้น |

## 2. หน่วยนับสินค้า (`Product Units`)

**ใช้ในส่วน:** ฟอร์มเพิ่ม/แก้ไขสินค้า, แสดงผลในระบบและหน้าร้าน  
**Field ในฐานข้อมูล:** `products.unit` (varchar)

| ลำดับ | หน่วยนับ | ตัวอย่างการใช้งาน |
| :---: | :--- | :--- |
| 1 | ชิ้น | สินค้าที่นับเป็นชิ้นๆ ทั่วไป |
| 2 | แพ็ค | สินค้าที่จัดเป็นชุด/แพ็ค เช่น แพ็ค 6 ชิ้น |
| 3 | โหล | สินค้าที่นับเป็นโหล (12 ชิ้น) |
| 4 | ลัง | สินค้ายกลัง เช่น ลัง 24 กล่อง |
| 5 | ขวด | สินค้าประเภทน้ำยา น้ำดื่ม บรรจุขวด |
| 6 | กระป๋อง | สินค้าบรรจุกระป๋อง เช่น สเปรย์ อาหารกระป๋อง |
| 7 | ห่อ | สินค้าที่ห่อด้วยฟิล์มหรือบรรจุภัณฑ์อ่อน |
| 8 | ถุง | สินค้าบรรจุถุง เช่น ถุงข้าว ถุงขนม |
| 9 | กล่อง | สินค้าที่บรรจุในกล่องกระดาษหรือกล่องแข็ง |
| 10 | คู่ | สินค้าที่ขายเป็นคู่ เช่น รองเท้า ถุงเท้า |
| 11 | ชุด | สินค้าที่ขายเป็นชุด เช่น ชุดเครื่องครัว |

## 3. สถานะคำสั่งซื้อ (`Order Status`)

**ใช้ในส่วน:** หน้าจัดการออเดอร์ (Admin), หน้าติดตามพัสดุ (ลูกค้า)  
**Field ในฐานข้อมูล:** `orders.status` (varchar)

| ค่าในระบบ (Value) | ชื่อแสดงผล (Label) | สี | ความหมาย |
| :--- | :--- | :--- | :--- |
| `PENDING` | รอตรวจสอบ | 🟡 เหลือง | ออเดอร์ใหม่ที่ยังไม่ได้รับการยืนยัน |
| `CONFIRMED` | รอจัดส่ง | 🔵 น้ำเงิน | แอดมินยืนยันแล้ว กำลังจัดเตรียมสินค้า |
| `SHIPPED` | กำลังจัดส่ง | 🟣 ม่วง | ส่งพัสดุให้ขนส่งแล้ว |
| `COMPLETED` | สำเร็จ | 🟢 เขียว | ลูกค้าได้รับสินค้าแล้ว ออเดอร์ปิด |
| `CANCELLED` | ยกเลิก | 🔴 แดง | ออเดอร์ถูกยกเลิก |

## 5. วิธีการจัดส่ง (`Delivery Method`)

**ใช้ในส่วน:** หน้าตะกร้าสินค้า (ลูกค้า), รายละเอียดออเดอร์  
**Field ในฐานข้อมูล:** `orders.deliveryMethod` (varchar)

| ค่าในระบบ (Value) | ชื่อแสดงผล | คำอธิบาย |
| :--- | :--- | :--- |
| `DELIVERY` | จัดส่งถึงบ้าน | ส่งพัสดุทางบริษัทขนส่ง |
| `PICKUP` | มารับเองที่ร้าน | ลูกค้ามารับสินค้าที่หน้าร้าน |

## 6. สิทธิ์ผู้ดูแลระบบ (`Admin Roles`)

**ใช้ในส่วน:** ระบบ Login, การแสดงเมนู Sidebar, การควบคุมฟีเจอร์  
**Field ในฐานข้อมูล:** `admins.role` (enum)

| ค่าในระบบ (Value) | ชื่อตำแหน่ง | สิทธิ์ที่มี |
| :--- | :--- | :--- |
| `ADMIN` | ผู้ดูแลระบบ | ดูและจัดการ Orders, Products, Reports, Dashboard |
| `SUPER_ADMIN` | ผู้ดูแลระบบสูงสุด | ทุกสิทธิ์ของ Admin + จัดการ Posts, Users, Settings |

---

## 7. ช่องทาง LINE (`LINE Integration`)

**ใช้ในส่วน:** ระบบแจ้งเตือน Order, LINE Login, LIFF  
**ค่าคงที่ระบบ (ตั้งค่าผ่าน Environment Variables)**

| ชื่อ ENV | คำอธิบาย | ตัวอย่าง |
| :--- | :--- | :--- |
| `LINE_CHANNEL_ACCESS_TOKEN` | Token สำหรับส่งข้อความผ่าน Messaging API | `Y4YhRd1X...` |
| `LINE_CHANNEL_SECRET` | Secret Key สำหรับยืนยัน Webhook | `8f8aab25...` |
| `LINE_LOGIN_CHANNEL_ID` | Channel ID สำหรับ LINE Login OAuth | `2006...` |
| `LINE_LOGIN_CHANNEL_SECRET` | Secret Key สำหรับ LINE Login | `abc123...` |
| `ADMIN_LINE_GROUP_ID` | Group ID LINE สำหรับรับแจ้งเตือน Order | `C4a9f3...` |
| `VITE_LIFF_ID` | LIFF ID สำหรับ LIFF App ลูกค้า | `2006678...` |
