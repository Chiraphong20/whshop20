# เอกสารการทดสอบระบบ (Test Cases)

เอกสารนี้รวบรวมกรณีการทดสอบ (Test Case) สำหรับระบบ E-Commerce (Wonghiranshop) เพื่อใช้สำหรับเป็นแนวทางตรวจรับงานระบบ หรือทดสอบหาข้อผิดพลาดก่อนเริ่มใช้งานจริง
